## 1. 事業概要

### 1.1 ミッション　

相続で不動産を取得した売主が、適正価格・短期間で売却でき、相続手続き全体をワンストップで解決できる日本初の相続不動産専門ポータルプラットフォームを提供する。

### 1.2 事業構造（三者モデル）

| プレイヤー | 役割 |
|-----------|------|
| **Ouver（自社）** | プラットフォーム運営、集客、マッチング、入札システム提供 |
| **提携宅建業者** | 重要事項説明・売買契約・決済の実務（宅建業法上の仲介行為） |
| **士業NW会社**（awaka cross / UIコンサルティング / ミツカル等） | 税理士・司法書士等のネットワーク提供、案件紹介の起点 |

### 1.3 重要な前提

- **Ouverは宅建業免許を持たない** → 仲介行為は行わない
- Ouverの収益は提携宅建業者からの**業務委託料**（集客・マッチング・入札運営の対価）
- ビジネスロジック（入札・問い合わせ・案件管理）は既にMVPとして実装済み

## 2. 相続ドメインの基礎知識

### 2.1 相続税の仕組み

| 項目 | 内容 |
|------|------|
| 基礎控除 | 3,000万円 + 600万円 × 法定相続人の数 |
| 税率 | 10%〜55%（累進課税） |
| 課税対象者 | 被相続人の約9.6%（年間約15万件） |
| 申告期限 | **相続開始から10ヶ月以内**（延長なし） |
| 納付方法 | 現金一括が原則（延納・物納は条件あり） |

### 2.2 相続税率テーブル

| 課税額 | 税率 | 控除額 |
|--------|------|--------|
| 〜1,000万円 | 10% | - |
| 〜3,000万円 | 15% | 50万円 |
| 〜5,000万円 | 20% | 200万円 |
| 〜1億円 | 30% | 700万円 |
| 〜2億円 | 40% | 1,700万円 |
| 〜3億円 | 45% | 2,700万円 |
| 〜6億円 | 50% | 4,200万円 |
| 6億円超 | 55% | 7,200万円 |

### 2.3 不動産の相続税評価

| 資産 | 評価方法 | 市場価格との関係 |
|------|---------|----------------|
| 土地 | 路線価方式 or 倍率方式 | 市場価格の約80% |
| 建物 | 固定資産税評価額 | 市場価格の約50〜70% |

→ **不動産は現金より相続税評価が低い**ため、節税対策で不動産を持つ人が多い。しかし、相続発生後に「現金化が必要」になるケースが本サービスのターゲット。

### 2.4 相続登記の義務化

- **2024年4月1日より義務化**
- 相続開始を知った日から**3年以内**に登記申請が必要
- 正当な理由なく怠った場合：**10万円以下の過料**
- 過去の相続にも遡及適用（2027年3月末まで猶予）

### 2.5 相続手続きのタイムラインと関与する士業

```
相続発生（被相続人の死亡）
  │
  ├─ 7日以内 ─── 死亡届提出
  │
  ├─ 3ヶ月以内 ── 相続放棄の判断
  │                └→ 弁護士（相続争いがある場合）
  │
  ├─ 4ヶ月以内 ── 準確定申告（被相続人の所得税）
  │                └→ 税理士
  │
  ├─ 遺産分割協議 ─ 相続人全員の合意
  │                  └→ 弁護士（争いがある場合）
  │                  └→ 行政書士（協議書作成）
  │
  ├─ 相続登記 ──── 不動産の名義変更
  │                └→ 司法書士 ★紹介の起点①
  │
  ├─ 10ヶ月以内 ── 相続税申告・納付 ★最重要デッドライン
  │                └→ 税理士 ★紹介の起点②
  │
  └─ 不動産売却 ── 現金化（納税資金確保 or 資産整理）
                    └→ 宅建業者（仲介）★Ouverが介入するポイント
```

### 2.6 仲介手数料の法定上限

| 売買価格 | 手数料率（税別） |
|---------|----------------|
| 200万円以下の部分 | 5% |
| 200万超〜400万円以下の部分 | 4% |
| 400万円超の部分 | 3% |
| **速算式（400万円超）** | **売買価格 × 3% + 6万円** |

**低廉な空家等の特例（2024年改正）：**
- 売買価格800万円以下の場合：片手につき上限**33万円**（税別）
- 売主・買主の双方から受領可能

## 3. ビジネスモデル

### 3.1 収益フロー

```
売主（相続人）
  │ 物件登録
  ▼
┌─────────────────────────────────┐
│  Ouver プラットフォーム          │
│  ・物件掲載・検索               │
│  ・入札（sealed bid）運営       │
│  ・マッチング                   │
└────────┬────────────────────────┘
         │ 入札成立 / マッチング成立
         ▼
  提携宅建業者（仲介実務）
         │ 重説→契約→決済
         ▼
  仲介手数料発生
         │
         ├→ 宅建業者: 50%
         ├→ Ouver:    32%（業務委託料として）
         ├→ 紹介士業:  15%（紹介料として）
         └→ NW会社:    3%（NW利用料として）
```

### 3.2 入札（sealed bid）方式の価値

| 観点 | 従来の仲介 | Ouverの入札方式 |
|------|-----------|---------------|
| 価格決定 | 仲介業者の査定（売主は受動的） | 複数買い手の競争入札で市場価格が形成 |
| 売却期間 | 3〜6ヶ月が一般的 | 入札期限を設定（2〜4週間） |
| 買い手の本気度 | 内見→交渉→離脱が多い | 入札=購入意思の表明 |
| 価格交渉 | 値下げ交渉が常態化 | 入札価格が最終提示 |

### 3.3 Ouverの法的ポジション

Ouverは宅建業法上の「仲介」を行わない。収益の名目は：

- **業務委託料**：集客・マッチング・入札運営システム提供の対価
- 宅建業者との間で業務委託契約を締結
- 仲介手数料を直接受領するのではなく、宅建業者が受領した手数料の一部を業務委託料として受け取る

## 4. ターゲット戦略

### 4.1 ターゲット価格帯

**Phase 1: 1,000万円以上の物件に限定**

| 価格帯 | 仲介手数料（両手） | Ouver取り分（32%） | 判定 |
|--------|------------------|------------------|------|
| 500万円 | 42万円 | 13.4万円 | ❌ 工数に見合わない |
| 1,000万円 | 72万円 | 23万円 | ✅ 最低ライン |
| 1,500万円 | 102万円 | 32.6万円 | ✅ 適正 |
| 2,000万円 | 132万円 | 42.2万円 | ✅ 理想 |
| 3,000万円 | 192万円 | 61.4万円 | ✅ 高収益 |

**1,000万円未満を除外する理由：**
- 各ステークホルダーの取り分が薄くなりすぎる
- 士業の紹介インセンティブとして機能しない
- 低価格帯は買取業者に流れる方が売主にとっても合理的

### 4.2 ターゲットエリア

**Phase 1: 東京都 + 神奈川県**

選定理由：

| 指標 | 東京都 | 神奈川県 |
|------|--------|---------|
| 相続税課税割合 | 16.3%（全国1位） | 13.0%（全国3位） |
| 不動産価格水準 | 高い（1,000万円+が多い） | 中〜高 |
| 買い手の密度 | 最も高い | 高い |
| 士業の集積度 | 最も高い | 高い |
| 相続発生件数 | 年間約12万件 | 年間約8万件 |

**Phase 2以降の拡大ロードマップ：**
1. Phase 2: 埼玉・千葉（首都圏完成）
2. Phase 3: 愛知・大阪（三大都市圏）
3. Phase 4: 全国主要都市

## 5. 士業アタック戦略

### 5.1 優先順位

| 優先度 | 士業 | 理由 |
|--------|------|------|
| **Tier 1** | 税理士 | 相続税申告（10ヶ月期限）の中心。不動産売却の意思決定に最も影響力がある |
| **Tier 1** | 司法書士 | 相続登記の実務担当。不動産を持つ相続人と必ず接点がある |
| Tier 2 | 弁護士 | 遺産分割争いの案件のみ。件数は限定的 |
| Tier 3 | 行政書士 | 遺産分割協議書作成。不動産売却への影響力は弱い |
| 対象外 | 不動産鑑定士 | 裁判・税務申告時の鑑定評価が主。紹介ルートとしては非効率 |

### 5.2 税理士へのアプローチポイント

- **相続税申告の中で「納税資金が足りない」ケースが紹介の起点**
- 税理士は不動産売却のプロではない → 信頼できる紹介先を求めている
- 紹介料（仲介手数料の15%）は税理士にとって追加収入になる
- 「顧客への付加価値サービス」として紹介しやすい

### 5.3 司法書士へのアプローチポイント

- **相続登記の依頼 → 「この不動産どうしますか？」が自然な会話の流れ**
- 2024年の登記義務化で相続登記の依頼件数が増加中
- 司法書士は不動産業者との接点が既にある → 既存の紹介先との比較になる
- Ouverの入札方式で「より高く売れる可能性」を訴求

### 5.4 NW会社の活用

| NW会社 | 特徴 |
|--------|------|
| awaka cross | 士業ネットワークの運営 |
| UIコンサルティング | 士業向けコンサルティング |
| ミツカル | 士業マッチングサービス |

- いずれかに限定せず、複数のNW会社と提携
- NW会社への支払い：仲介手数料の**3%**
- NW会社の役割：士業の紹介・取りまとめ、初期アプローチの支援

## 6. 紹介インセンティブ設計

### 6.1 収益配分モデル（確定版）

| ステークホルダー | 配分率 | 1,500万円物件での金額 | 役割 |
|----------------|--------|---------------------|------|
| 提携宅建業者 | 50% | 51万円 | 仲介実務（重説・契約・決済） |
| Ouver | 32% | 32.6万円 | プラットフォーム運営・集客・入札 |
| 紹介士業 | 15% | 15.3万円 | 案件紹介（税理士 or 司法書士） |
| NW会社 | 3% | 3.1万円 | 士業ネットワーク提供 |
| **合計** | **100%** | **102万円** | |

※ 仲介手数料の両手（売主側+買主側）合計を100%として配分

### 6.2 士業の紹介料の位置づけ

- 税理士・司法書士への紹介料は法的に**明確な禁止規定がない**
- ただし、各士業団体の倫理規程に配慮が必要
- 支払名目は「情報提供料」「業務協力金」等が一般的
- 支払元はOuver（業務委託料の中から）or 宅建業者（仲介手数料の中から）

### 6.3 各ステークホルダーの損益分岐

**紹介する士業の視点：**
- 1件あたり15.3万円（1,500万円物件の場合）
- 月1件紹介すれば年間約180万円の追加収入
- 本業の顧客サービス向上にもつながる

**提携宅建業者の視点：**
- 1件あたり51万円（1,500万円物件の場合）
- 広告費ゼロ、営業工数は重説〜決済のみ
- 自社集客の案件（手数料100%）より単価は低いが、純増の売上

**Ouverの視点：**
- 1件あたり32.6万円（1,500万円物件の場合）
- 固定費（サーバー・人件費等）を月6件の成約でカバー

## 7. 提携宅建業者の交渉戦略

### 7.1 提携業者に求める要件

| 要件 | 必須/希望 |
|------|----------|
| 宅建業免許（知事または大臣） | 必須 |
| 相続案件の実務経験 | 必須 |
| 東京・神奈川で営業可能 | 必須 |
| 少人数・中小規模（社員5〜15名） | 強く希望 |
| ITリテラシー（プラットフォーム対応） | 希望 |
| 買取再販の能力（入札不成立時のバックアップ） | 希望 |

### 7.2 業者へのメリット訴求

| メリット | 詳細 |
|---------|------|
| 集客コストゼロ | 自社で広告を打たなくても案件が来る |
| 案件の質が高い | 士業経由＝相続手続きが進んでいる売主。冷やかしが少ない |
| 成約率が高い | 入札方式＝買い手が見つかった状態で仲介に入れる |
| 実務に集中 | 集客・マーケティングはOuverが担当 |
| カニバリなし | 既存ビジネスを食うのではなく、純増の売上 |

### 7.3 手数料交渉の段階的アプローチ

業者が50%配分に難色を示した場合の段階移行モデル：

| フェーズ | 業者 | Ouver | 士業 | NW | 条件 |
|---------|------|-------|------|-----|------|
| 最初の5件 | 60% | 22% | 15% | 3% | 実績作り期間 |
| 6〜20件 | 55% | 27% | 15% | 3% | 安定供給の証明後 |
| 21件〜 | 50% | 32% | 15% | 3% | 本来条件に移行 |

### 7.4 片手仲介への対応

買い手が自分の仲介業者を連れてくるケースでは仲介手数料が片手になる。

- 片手の場合は業者の取り分を**60%**に引き上げる補正条項を設ける
- Ouverの買い手プールから入札させることで**両手仲介を基本設計**とする

### 7.5 業者探しのアプローチ方法

| 方法 | 有効度 | 詳細 |
|------|--------|------|
| 司法書士からの紹介 | ★★★★★ | 相続登記で普段紹介している不動産屋。既に相続案件に慣れている |
| NW会社経由 | ★★★★☆ | 士業NWに不動産業者も含まれている可能性 |
| Google検索 | ★★★☆☆ | 「相続 不動産 仲介 東京」でSEO投資している業者 |
| 宅建協会名簿 | ★★☆☆☆ | エリアで絞り込み、地域密着型をリストアップ |

### 7.6 契約の主要条項

| 条項 | 内容 |
|------|------|
| 業務範囲 | Ouver：集客・マッチング・入札 / 業者：重説・契約・決済 |
| 報酬配分 | 仲介手数料の○%を業務委託料としてOuverに支払い |
| 支払時期 | 決済完了後○営業日以内 |
| 非独占 | 業者は他の集客チャネルも自由に使える |
| 案件拒否権 | 業者は案件ごとに対応可否を判断できる |
| 品質基準 | 対応品質に関する最低基準 |
| 契約期間 | 1年更新、3ヶ月前通知で解約可能 |
| 秘密保持 | 案件情報・顧客情報の取り扱い |

## 8. 競合分析・差別化戦略

### 8.1 競合マップ

```
                    相続特化度 高い
                         ▲
                         │
   不動産相続相談窓口 ●   │        ● Ouver（目指すポジション）
   (大阪限定・買取型)     │          (ポータル+入札+士業NW)
                         │
   訳あり物件買取プロ ●   │
   (問題物件全般)         │
                         │
   ─────────────────────┼──────────────────────►
   単機能                │              サービス統合度 高い
   (査定/買取のみ)        │
                         │
        空き家バンク ●    │    ● リビンマッチ
        (自治体・無料)    │      (一括査定)
                         │
             楽待 ●      │         ● SUUMO/アットホーム
          (投資家向け)    │           (総合ポータル)
                         │
                    相続特化度 低い
```

### 8.2 競合別分析

| 競合 | 概要 | 強み | Ouverの優位点 |
|------|------|------|-------------|
| **不動産相続相談窓口** | 大阪拠点、相続特化の買取/仲介 | 相続特化、司法書士提携 | エリア（東京）、入札方式、ポータル機能 |
| **リビンマッチ** | TSE上場、一括査定サービス | 2,100社NW、知名度、年間24万件査定 | 相続特化、士業NW、営業電話なし |
| **楽待** | 最大手投資物件ポータル（11万件） | 圧倒的な買い手（投資家）プール | 売主目線の設計、適正価格（買い叩かれない） |
| **訳あり物件買取プロ** | 問題物件専門の買取 | 問題物件対応力 | 市場価格での売却（買取の50-70%より高い） |
| **空き家バンク** | 自治体運営、899市区町村参加 | 自治体信用、無料 | スピード、価格帯（1,000万+）、成約率 |
| **SUUMO/アットホーム** | 総合不動産ポータル | 圧倒的集客力 | 相続文脈のUX、士業連携、入札方式 |
| **税理士法人チェスター** | 相続税申告最大手（7,000件/年） | 顧客基盤 | 不動産売却の専門性、プラットフォーム |

### 8.3 Ouverの5つの差別化ポイント

**① 「相続 × 不動産」完全特化ポータル**
- 相続特有の情報（手続き状況、売却緊急度、相続登記の有無）を物件情報に統合
- 買い手にとっても「相続物件=交渉しやすい」という明確なメリット

**② 入札（sealed bid）方式**
- 競争原理による価格最適化（買取の50-70%ではなく市場価格で成約）
- スピードと価格の両立（入札期限設定で短期間に決着）
- 他の競合で入札方式を採用しているサービスは存在しない

**③ 士業NWからのリードパイプライン**
- 広告ではなく士業からの直接紹介 → CAC（顧客獲得コスト）ほぼゼロ
- NW構築には時間がかかる → それ自体が参入障壁

**④ 売主の困りごとワンストップ対応**

| 困りごと | SUUMO | リビンマッチ | 買取業者 | Ouver |
|---------|-------|-----------|---------|-------|
| 相続登記がまだ | ❌ | ❌ | ❌ | ✅ 司法書士紹介 |
| 相続税申告が不安 | ❌ | ❌ | ❌ | ✅ 税理士紹介 |
| 遺産分割が未了 | ❌ | ❌ | ❌ | ✅ 弁護士紹介 |
| 適正価格不明 | △ | △ | ❌ | ✅ 入札で価格形成 |
| 早く現金化したい | ❌ | △ | ✅ | ✅ 入札期限設定 |
| 営業電話が嫌 | - | ❌ | ❌ | ✅ PF経由 |

**⑤ 10ヶ月タイムリミットの活用**
- 相続税申告期限をUXに組み込んだ設計（緊急度表示、期限カウントダウン）
- 税理士が「申告と一緒に売却も」と自然に紹介できる文脈

### A. 用語集

| 用語 | 意味 |
|------|------|
| 両手仲介 | 売主・買主双方の仲介を1社が行うこと |
| 片手仲介 | 売主 or 買主の一方のみを仲介すること |

| 重説 | 重要事項説明。宅建士による法定の説明義務 |
| CAC | Customer Acquisition Cost（顧客獲得コスト） |
| NW | ネットワーク |
